## вҡҷпёҸ Installation

Clone the repository:

```bash
git clone https://github.com/Apurv3027/Tirth-Darshan.git
```

Go to project folder:

```bash
cd Tirth-Darshan
```

Install dependencies:

```bash
npm install
```
